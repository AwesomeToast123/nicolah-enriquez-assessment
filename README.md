# Airport Spotter

A mini-app project

## Getting Started 

This an app that collects airport info from all over the world and its details 
specifically from the following:

   - id
   - gmt
   - airportId
   - iataCode
   - cityIataCode
   - icaoCode
   - countryIso2
   - geoNameId
   - latitude
   - longitude
   - airportName
   - countryName
   - phoneNumber
   - timezone

## setting up

- Download the latest flutter sdk
- Download the flutter plugin for compatibility
- set the ndk version to  27.0.12077973 in your android gradle  android/app/build.gradle.kts (e.g ndkVersion = "27.0.12077973")
- type flutter clean 
- type flutter pub get to get the latest packages
- type flutter run or any button or command(flutter run) that makes the app run
- create any emulator (preferably with 8gb to run this)
